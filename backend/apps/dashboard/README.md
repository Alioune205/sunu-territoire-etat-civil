# Dashboard App

Cette application est réservée à l'équipe **DEV 2**.
Elle gérera les agrégations de données et les statistiques pour le dashboard React des administrateurs.

## Responsabilités
- Endpoints de KPIs (Total dossiers, statuts, temps de traitement moyen)
- Endpoints de graphiques (Évolution des demandes, répartition par commune)
- Export CSV/Excel des données
