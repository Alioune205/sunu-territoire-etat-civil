# AI App

Cette application est réservée à l'équipe **DEV 2**.
Elle gérera l'intelligence artificielle, notamment l'OCR pour vérifier les documents.

## Responsabilités
- Analyse d'image OCR (extraction de texte depuis les scans)
- Vérification intelligente de la cohérence des données (CNI vs Profil)
- Mise à jour automatique des statuts OCR sur le modèle `Document`
