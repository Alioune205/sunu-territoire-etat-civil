# Notifications App

Cette application est réservée à l'équipe **DEV 2**.
Elle gérera l'envoi et le suivi des notifications (Email, SMS, Push) aux citoyens et agents.

## Responsabilités
- Modèle `Notification`
- Intégration Firebase Cloud Messaging (FCM) ou service d'emails
- Endpoints pour récupérer l'historique des notifications
