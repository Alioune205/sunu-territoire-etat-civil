# Assets - Seals Directory

Place your SVG/PNG files here:

- `cachet_communal.svg` — Cachet officiel de la commune (cercle avec armoiries)
- `signature_officier.svg` — Signature de l'officier d'état civil

## Formats supportés
- SVG (recommandé pour la meilleure qualité d'impression)
- PNG (avec transparence)

## Notes
Si les fichiers SVG ne sont pas présents, le système générera des placeholders circulaires en pointillés.
